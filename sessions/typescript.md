# Intro to TypeScript
TypeScript stands in an unusual relationship to JavaScript. TypeScript offers all of JavaScript’s features, and an additional layer on top of these: TypeScript’s type system.

## Goal of learning
By completing this tutorial, you will be able to:
- Understand below fundmentals of TypeScript
    - TypeScript type basics
    - TypeScript for OOP programmers
    - TypeScript for functions

### More on Functions
Functions are the basic building block of any application, whether they’re local functions, imported from another module, or methods on a class. They’re also values, and just like other values, TypeScript has many ways to describe how functions can be called. Let’s learn about how to write types that describe functions. Read below article to learn more about TypeScript and functions.

[More on Functions](https://www.typescriptlang.org/docs/handbook/2/functions.html)
